Herzlich Willkommen zum ECP - LiveTicker Mobile V0.93 AWM

Bitte beachten Sie, bei der Software/Webseite handelt es sich um ein Projekt im Rahmen meines Informatikstudiums. Es kann also in der Software zu gelegentlichen Fehlern kommen, oder bestimmte Features nicht verfügbar sein. Ich werde mich bemühen bei den kommenden Updates die Fehler zu beheben. Sollten Sie einen Fehler bemerken, so nutzen Sie die Möglichkeit "Fehler per Email melden!" um mich auf einen Fehler hinzuweisen.

Diese Version ist eine Webseite für mobile Entgeräte, sie wurde mit Hilfe von jQuery Mobile erstellt.

Sie können den Ticker auch auf einem iOS Gerät verfolgen. Für iPhone, iPadund iPod touch haben wir eine eigene App entwickelt. Diese App "iLiveTicker" finden Sie hier im AppStore.

Vielen Dank und viel Spaß mit dem Ticker,
